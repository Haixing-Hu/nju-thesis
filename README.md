# NJU-Thesis

## 定制说明

这是由 Zachary Cao 在 Haixing Hu 原项目上的个人定制模板，除了对原项目的一些 bug 做了修复，也根据个人审美和一些统一性要求对模板的作出改动，包括：

* 修复了论文短标题无法正常显示的错误；
* 版面设计：
    1. 添加中英文摘要纸到目录；
    2. 中英文摘要纸字样双下划线;
    3. 中英文摘要纸自动去掉多余的空行;
    4. 修改标点符号处理方式为全角；
    5. 计算了等效 Word 1.5 倍行距的缩放系数；
* 字体字号：
    1. 修正了公式内数字不是 Times 字体的错误；
    2. 新增了 fandol 开源字体，使得粗体不必用黑体代替；
    3. 修改了各字号的字体大小，不再对磅 (Word) 和 pt (TeX) 进行换算；
    4. 修改页眉中文字体为楷书；
* 字段信息：
    1. 使用新的南大官方高清 logo 替换了原 logo；
    2. 封面和授权书表格信息字段修改；
    3. 修改了生成pdf文件的主题字段;
    4. 使用长论文标题时，也可以正确生成 pdf 文件标题信息;
* `sample.tex` 文件中文字体使用 fandol 字体。
* 修改 `Makefile` 文件，解决了 win 平台 texlive 2018 无法正确编译 `.ins` 的问题。


*********

本项目提供了一个用于排版南京大学学位论文的XeLaTeX模板。该模板严格遵循南京大学学术委员会对学位论文的排版格式要求以及相应的国家标准规范，可帮助南京大学的同学们方便地排版出漂亮的学位论文。

目前该模板支持排版博士、硕士和学士的学位论文。

虽然该模板主要用于排版南京大学的学位论文，但亦可经过简单的设置或修改用于排版国内其他大学的学位论文。

## 功能特色

* 此模版用于生成符合南京大学学位论文排版要求和相应的国家规范、行业标准的学位论文；
* 可排版博士、硕士、学士学位论文；
* 使用XeLaTeX作为排版引擎，论文源码需要使用UTF-8编码；
* 自动生成国家图书馆封面、中文封面、英文封面、中文摘要页、英文摘要页、学位论文出版授权书等必需页面；
